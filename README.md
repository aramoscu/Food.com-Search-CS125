# Food.com-Search-CS125
To run:<br>
- Download RAW_interactions.csv and RAW_recipes.csv from Kaggle Dataset: https://www.kaggle.com/datasets/shuyangli94/food-com-recipes-and-user-interactions?select=RAW_recipes.csv<br>
- Move both files into the inverted_index directory<br>
- Run build_inverted_index.py to build the inverted index
- Run search.py to perform the search demo
- The Parameters of the example queries can be changed manually

Raw_recipes.csv: Recipes<br>
Raw_interactions.csv: Recipe Reviews<br>
